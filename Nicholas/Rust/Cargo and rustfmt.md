# Cargo and rustfmt
***
###### tags: #Программирование/Rust 
***
Creating a new rust project:
```
cargo new "folder name"
```
Compiling code:
```
cargo build
cargo run
```
This makes our complier check all possible easy catchable bugs for us
```
cargo check
```
File formating, i.e. deleting excessibe spaces, tabs etc.
```
rustfmt main.rs
```
