# Cargo и rustfmt
Создание нового проекта в Rust 

	cargo new "folder name"
Компилирование кода в Rust 

	cargo build
	cargo run

"Проверка" кода

	cargo check
Форматирование файла, "убирает лишние пробелы и т.д."

	rustfmt main.rs
